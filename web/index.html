指标阅读说明
mountain_CBI 表示单条线性山脉的综合文化分隔强度； local_signal_CBI 表示该山脉是否存在较强局部分异信号； reliable_coverage_ratio 表示可靠窗口覆盖比例； high_CBI_coverage_ratio 表示高 local_CBI 窗口覆盖比例； contrast_segment_ratio 表示地名文化社区分异连续段覆盖比例； evidence_quality 用于判断结果是否可以正式解释。
  • 局部窗口颜色深，不等于整条山脉 mountain_CBI 一定高。
  • mountain_CBI 应结合 rank、coverage 和 evidence_quality 综合判断。
  • 横断山这类复合山系，应同时参考下方 mountain_system_CBI 山系补充视图。

地图阅读方式

  • 彩色粗线表示不同移动窗口，颜色越深 local_CBI 越高。
  • 金色线表示地名文化社区分异连续段。
  • 圆点表示被该山脉分隔的地名特征点。
  • 点击窗口可查看 CBI、small_CBI、big_CBI、可靠性和 clip 风险。
  • 点击连续段可查看平均 segment_local_CBI 和主导社区。
  • 点击地名点可查看地名、特征字、small_feature 和 big_feature。
  • 窗口图例只解释 local_CBI，不等于 mountain_CBI 的山脉整体等级。

窗口与侧宽分析

沿山脉 station 的 local_CBI 曲线

横坐标：沿山脉方向的 station 位置，单位为 km；纵坐标：移动窗口 local_CBI，表示该窗口内山脉两侧地名文化社区结构差异强度。

small_CBI / big_CBI 对比

横坐标：沿山脉方向的移动窗口 station 位置，单位为 km；纵坐标：CBI 指数值。small_CBI 表示地名文化社区小类差异，big_CBI 表示语义大类差异。

侧宽敏感性 mountain_CBI / local_signal

横坐标：山脉两侧统计宽度，单位为 km；纵坐标：归一化后的指标值。柱状值表示可靠覆盖或覆盖比例，折线值表示不同侧宽下的 mountain_CBI / local_signal 变化。

地名文化社区分异连续段

横坐标：分异连续段的 local_CBI 平均值，数值越大表示该连续段内山脉两侧地名文化社区差异越强；纵坐标：地名文化社区分异连续段编号。
